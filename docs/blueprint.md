# **App Name**: Pithampur Property Hub

## Core Features:

- User Authentication: User authentication via Firebase Auth (phone OTP or email). Sellers must sign in to list; buyers can browse but need to sign in to contact sellers.
- Property Listing Form: Property listing form for sellers with fields for property type, transaction, location (Google Maps integration), price, area, bedrooms/bathrooms, description, photos (up to 10), amenities, and contact info.
- Listing Review: Review screen for sellers to confirm listing details before publishing.
- Listing Display: Display recent and featured listings on the home/dashboard page for buyers.
- Search & Filter: Search and filter functionality for buyers to find listings based on property type, sale/rent, price range, area range, and locality.
- Listing Detail: Listing detail page with full images (carousel), description, amenities, seller info, and call/WhatsApp seller buttons.

## Style Guidelines:

- Primary color: Light cyan (#A0CED9) for a fresh and trustworthy feel.
- Background color: Very light cyan (#F0FAFB), near-white.
- Accent color: Light greenish-yellow (#BCD9A0) for a subtle call to action.
- Body and headline font: 'Inter' sans-serif, for a clean, modern, neutral look.
- Simple, clear icons for property types and amenities.
- Responsive layout adapting to web and mobile screens.
